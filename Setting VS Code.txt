// Перелік плагінів:

1. advanced-new-file;  
// Плагін дозволяє швидко створювати нові файли та папки в будь-якому місці проекту без необхідності вручну переходити в потрібну директорію. Він економить час, дозволяючи одразу задавати шлях і назву нового файлу через командний рядок VS Code.

2. All Autocomplete;  
// Плагін розширює стандартну автодоповнення VS Code, скануючи всі відкриті файли проекту для пропозицій. Це зручний інструмент для швидкого доступу до змінних, функцій або інших ключових слів, які використовуються в різних файлах, покращуючи продуктивність і зменшуючи кількість помилок під час кодування.

3. Auto Close Tag;  
// Плагін автоматично закриває HTML та XML теги, коли ви відкриваєте їх. Це зменшує кількість ручної роботи та допомагає уникнути помилок у синтаксисі, забезпечуючи швидке і точне написання тегів у мовах розмітки.

4. Auto Complete Tag; 
// Плагін автоматично завершує відкриті теги, додаючи закриваючий тег одночасно з відкриттям. Це дуже зручно при написанні HTML, XML або подібних мов розмітки, оскільки прискорює роботу і допомагає уникнути помилок через забуті закриваючі теги.

5. Auto Rename Tag;  
// Плагін автоматично перейменовує парні HTML або XML теги одночасно. Якщо ви змінюєте назву відкриваючого тегу, закриваючий тег змінюється автоматично. Це зручно для підтримання коректної структури розмітки та уникнення синтаксичних помилок.

6. Autoprefixer;  
// Плагін автоматично додає вендорні префікси до CSS властивостей, щоб забезпечити кросбраузерну підтримку стилів. Він аналізує ваш CSS і додає префікси тільки там, де це потрібно, спрощуючи процес оптимізації стилів для різних браузерів і зменшуючи кількість ручної роботи.

7. Babel JavaScript; 
// Додає підтримку Babel для сучасного JavaScript, включаючи JSX та ES6+, з покращеною підсвіткою синтаксису та іншими функціями для зручнішої роботи.

8. Beautify css/sass/scss/less; 
// Форматує CSS, SASS, SCSS, та LESS код, забезпечуючи його акуратність і читабельність.

9. Bracket Pair Colorization Toggler; 
// Підсвічує відповідні дужки різними кольорами, що робить код більш структурованим і зрозумілим.

10. Class autocomplete for HTML; 
// Пропонує автодоповнення класів з ваших стилів у файлах HTML, що прискорює верстку та зменшує помилки.

11. className Completion in CSS; 
// Забезпечує автодоповнення імен класів у CSS та JavaScript файлах, зокрема для JSX та className.

12. Code Runner; 
// Дозволяє запускати код на різних мовах (JavaScript, Python, PHP тощо) прямо з редактора, що спрощує тестування.

13. Color Info; 
// Відображає інформацію про кольори у вашому коді, включаючи їх візуальне представлення та коди кольорів.

14. CSS Navigation; 
// Дозволяє швидко переходити між CSS правилами та їх використанням у HTML файлах.

15. CSScomb; 
// Інструмент для впорядкування CSS властивостей у стилях за заданими правилами, покращуючи читабельність і структуру.

16. eCSStractor for VSCode; 
// Витягує CSS властивості з HTML файлів, дозволяючи швидко створювати стилі на основі розмітки.

17. EditorConfig for VS Code; 
// Зчитує файли .editorconfig для налаштування редактора відповідно до загальних параметрів проекту.

18. GitLens — Git supercharged
// Додає візуальне відстеження змін у коді: хто, коли і навіщо змінив кожен рядок, прямо в редакторі.

19. Highlight Matching Tag; 
// Підсвічує відповідні відкриваючі та закриваючі теги в HTML і XML, полегшуючи навігацію та виправлення помилок.

20. HTML CSS Support; 
// Додає автодоповнення для CSS у HTML файлах, допомагаючи швидше працювати зі стилями без необхідності перемикатися між файлами.

21. HTML to CSS autocompletion; 
// Автодоповнення для CSS, що базується на HTML елементах, дозволяє швидко застосовувати стилі до елементів.

22. Image preview; 
// Показує попередній перегляд зображень у вашому коді, коли ви наводите курсор на шлях до файлу зображення.

23. IntelliSense for CSS class names in HTML; 
// Пропонує автодоповнення імен класів із CSS у HTML файлах, спрощуючи роботу з розміткою та стилями.

24. JavaScript Debugger (Nightly); 
// Остання нічна версія дебагера для JavaScript, що забезпечує покращений досвід налагодження коду.

25. Live Server; 
// Запускає локальний сервер і забезпечує живе оновлення сторінки в браузері при збереженні файлів, що спрощує процес розробки.

26. Lorem ipsum; 
// Генерує текст "Lorem ipsum" для використання як заглушку при дизайні веб-сторінок.

27. Material Icon Theme; 
// Змінює іконки файлів і директорій у VS Code.

28. "One Dark Pro"
// Тема для редактора. Рекомендується в цій темі обрати "One Dark Pro"

29. open in browser; 
// Додає можливість відкривати HTML файли безпосередньо у браузері з VS Code.

30. Path Autocomplete; 
// Пропонує автодоповнення шляхів до файлів і папок, що полегшує навігацію та вставку шляхів у коді.

31. Path Intellisense; 
// Розширює автодоповнення для шляхів до файлів у проекті, що прискорює написання коду.

32. Prettier - Code formatter; 
// Автоматично форматує ваш код відповідно до стандартів Prettier, забезпечуючи чистоту та читабельність коду.

33. px to rem & rpx & vw (cssrem); 
// Конвертує значення px у rem, rpx, або vw, що допомагає створювати адаптивні стилі.

34. Start git-bash; 
// Дозволяє запускати Git Bash безпосередньо з VS Code для керування версіями за допомогою Git.

35. Svg Preview;
// Показує попередній перегляд SVG файлів прямо в редакторі, що зручно для швидкої перевірки графіки.

36. CSS Peek
// Допомагає швидко перейти з html до css при затисканні Ctrl та натисканні по назві класу в html документі.

37. PHP Intelephense
// Додає розумну підсвітку синтаксису PHP, автодоповнення, перехід до визначення, підказки по PHPDoc, аналіз коду та багато іншого.

38. PHP CS Fixer
// Розширення для автоматичного форматування PHP-коду за допомогою правил стилю (PSR-12, Symfony тощо).
// Після встановлення, бажано зробити його глобально за допомогою команди: composer global require friendsofphp/php-cs-fixer

39. Code Spell Checker
// Легкий, але потужний плагін для VS Code, який перевіряє орфографію в коді — змінні, коментарі, рядки.

40. Ukrainian - Code Spell Checker
// Словник української мови для редактора.

41. Russian - Code Spell Checker
// Словник російської мови для редактора.

42. Tailwind CSS IntelliSense
// Мастхев плагін для VS Code, якщо працюєш з Tailwind CSS. Робить твоє життя простішим і швидшим.

43. Font Awesome Auto-complete & Preview
// Цей плагін додає автодоповнення та прев’ю іконок Font Awesome прямо в редакторі. Працює з HTML, JSX, Vue тощо.

44. JavaScript (ES6) code snippets
// Цей плагін додає набір готових сніпетів для сучасного JavaScript (ES6+) у VS Code

45. Multiple clipboards for VSCode
// Розширює стандартний буфер обміну VSCode до безсмертного багажа з копіпастами.

46. SCSS Formatter
// Автоматично вирівнює, структуризує та чистить SCSS-код, щоб він був читабельним і послідовним.
